Visión del Proyecto

Integrar el desarrollo de aplicaciones móviles y web en un ecosistema profesional. Buscamos transformar conceptos en sistemas reales mediante el dominio de tecnologías modernas, arquitecturas escalables y metodologías ágiles.

Ruta de Aprendizaje

Cada clase construye sobre la anterior. Seguí el hilo — cada concepto que aprendés hoy es un bloque del sistema que vas a construir mañana.

🗄️
2° Año CLASE 01

Backend con PHP & MySQL

El punto de partida. Variables, tipos de datos, operadores y decisiones en PHP. Conexión a MySQL con XAMPP, CRUD completo con formularios (Crear · Leer · Actualizar · Eliminar), estructura del lenguaje SQL, Primary Keys, Foreign Keys e integridad referencial. Acá nace la lógica de negocio de todo sistema.

Ir a la clase →

💡 El servidor PHP que construiste en clase 01 va a ser la API que consume la app móvil. El backend no cambia — solo cambia quién lo llama.

📱
2° Año CLASE 02

Apps Móviles con React Native

Con el backend funcionando, damos el salto al frontend móvil. React Native + Expo nos permiten crear apps iOS y Android reales. El servidor PHP del paso anterior responde en JSON — la app lo consume con fetch().

Ir a la clase →

🚀 En 3° año vas a construir la web app que comparte la misma API que la app móvil de 2°. PHP → JSON → React Native (móvil) y React (web) al mismo tiempo. Un solo backend, dos frontends.

🏗️
3° Año CLASE 01
